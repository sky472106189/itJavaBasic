- `XML`文件中左右尖括号有特殊含义，想要表示其本身，需要使用特殊写法

    | 符号 | 书写方式 | 备注                      |
    | ---- | -------- | ------------------------- |
    | `<`  | `&lt;`   | 小于符号（less then）     |
    | `>`  | `&gt;`   | 大于符号（greatter then） |
    | `"`  | `&quot;` | 双引号                    |
    | `'`  | `&apos;` | 单引号                    |
    | `&`  | `&amp;`  | 与符号                    |


- `XML`文件中可以存在CDATA区，该区中内容不会被XML解析，会被当做普通文本。格式：
  ```xml
  <![CDATA[这里的内容会被当做普通的字符串解析，比方说< > & ' "]]>
  ```
